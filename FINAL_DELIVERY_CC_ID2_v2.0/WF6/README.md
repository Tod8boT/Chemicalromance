# WF6: Nano Banana Image Edit System

AI-powered object replacement for product mockups.

## Features

- Replace mock objects with real CREMO products
- Perspective and lighting matching
- Product catalog (2 CREMO freezers)
- Google Drive integration

## Quick Start

1. Import `Image_Edit_Control.json`
2. Configure Nano Banana API key
3. Send `/edit_image` to bot
4. Select product → Upload image → Get edited result
